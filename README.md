## AI For Good - Introduction to Terratorch (19.03.2026)

Material for the AI for Good webinar "**Introduction to Terratorch**". 

The webinar introduces the TerraTorch library, covering the motivations for its development, its architectural design, and practical use cases. 
It also features hands-on demonstrations that showcase real-world case scenarios using TerraTorch. The material in this repository will guide participants in setting up a new experiment, exploring key functionalities, and highlighting the library's potential applications. 

The hands-on session is split into two parts:
- A demonstration line by line on how to set up a GFM pipeline with TerraTorch;
- A live coding example showing where to look for information in the TerraTorch documentation, giving participants the possibility to practice.
